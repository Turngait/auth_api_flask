# Auth API
## powered by Flask
### v. 0.1a4

Template AuthAPI on Flask for my own pet-projects.

## Requirements
```
git
docker
docker-compose
```

## Local setup
```
git clone https://github.com/Turngait/auth_api_flask.git auth_api
cd auth_api
cp .env.example .env
```
Fill all fields in .env .

```
docker-compose build
docker-compose up
```
